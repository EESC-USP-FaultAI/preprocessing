CSV Processing (functions.CSVProcessing)#

This module is used to process the CSV files. It contains two functions:

data_selection(data_file_path, meas[, ...])

Select the voltage/current data given its name on file "medicoes.csv".

case_selection(data_folder_path[, meas, ...])

Select the voltage/current data given the scenario parameters.
