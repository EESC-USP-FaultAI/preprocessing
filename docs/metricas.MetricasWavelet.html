metricas.MetricasWavelet.metricas_yordanos(s, level=1, wavelet='db4', phase=1)#

This function calculates several statistical measures for a given signal. The signal is first decomposed using the Discrete Wavelet Transform (DWT).

Parameters: - s (matrix-like): The signal to be analyzed. - level (int): The level of decomposition in the DWT. Default is 1. - wavelet (str): The name of the wavelet to be used in the DWT. Default is ‘db4’.

Returns: - list: A list containing the calculated metrics for the signal. The metrics are as follows:

  • F1: Skewness of the signal.

  • F2: Mean of the signal.

  • F3: Energy of the signal, calculated using the sum of squares method.

  • F4: Entropy of the signal.

  • F5: Standard deviation of the signal.

  • F6: Kurtosis of the signal.
